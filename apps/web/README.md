# WA OS Web

Interface web responsiva do WA OS, construída com Next.js, TypeScript e Tailwind CSS.

## Requisitos

- Node.js 20+
- npm 10+

## Desenvolvimento

```bash
npm install
npm run dev
```

A aplicação fica disponível em `http://localhost:3000`.

## Verificações

```bash
npm run lint
npm run typecheck
npm run build
```

O dashboard atual usa dados demonstrativos determinísticos enquanto a API Laravel é conectada. Ele inclui o shell responsivo, indicadores, atividade, acompanhamento operacional e campanhas recentes.
